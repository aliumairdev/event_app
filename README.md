Event App
================

## Ruby on Rails

This application requires:

- Ruby 2.4.0
- Rails 5.2.3

Learn more about [Installing Rails](http://railsapps.github.io/installing-rails.html).

## Getting Started

```bash
git clone https://github.com/m-huzaifa/event_app.git
cd event_app
```

## Setup

```shell
bin/setup
```
Run this again if you want a clear instance of **database**.

## Server

```shell
rails s
```
## Run Test Suite/ Rspec

```shell
rspec
```
**Specs covers following test cases:**

- Models specs
- Routing specs
- Views specs
- Requets specs
